# 🌤 Weather Now

A simple React app to check real-time weather conditions for any city.

## 🚀 Features
- Search weather by city
- Light/Dark mode toggle
- Displays temperature, condition, and timestamp

## 🛠 Tech Stack
- React
- Plain CSS
- Open-Meteo API

## 📦 How to Run
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
